## Getting Started

### To get a local copy run the following steps:

Please get your browser update.

. Step 1:  To get a local copy up and running follow these simple steps.
   ```
   git clone <URL> in your terminal
   ```

. Step 2: 
   ```
   cd metrics-app-covid19

   npm install

   npm run build

   npm start 
   ```

. Step 3: Test 
   ```
   npm test 
   ```
